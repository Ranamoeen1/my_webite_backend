Platforms like YouTube and Instagram may block automated requests. To fix this:

1. Export cookies from your browser using the [Get cookies.txt](https://chrome.google.com/webstore/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc) extension
2. Upload `cookies.txt` to this Space's Files tab
3. The backend will automatically use it for authenticated downloads
